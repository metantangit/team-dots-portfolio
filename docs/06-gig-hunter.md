# Gig Hunter - 프리랜서 일감 자동 헌팅 시스템

## 한줄 요약
당근마켓/숨고에서 IT/AI 일감을 AI로 자동 분류하고 즉시 지원하는 프리랜서 일감 헌팅 봇

## 프로젝트 개요
- **플랫폼**: 당근마켓 (라이브), 숨고 (계획중)
- **커버리지**: 전국 73개 지역
- **스캔**: 하루 3-4회, 12시간 윈도우
- **전략**: 고가치 기회 감지 → 빠른 지원 → 업셀링

## 핵심 기능

### 당근마켓 스캐너
- **전국 73개 지역 커버**: 서울 25구, 경기 12시, 주요 광역시
- **12시간 윈도우**: 시간당 조절 가능
- **예상 처리량**: ~12,000건/스캔 → 20-30건 키워드 매칭 → 5-10건 AI 리뷰

### 2단계 AI 분류

**Level 1: 키워드 매칭**
- 포함: AI, 인공지능, 코딩, Python, 개발자, 자동화, ChatGPT, 영상편집...
- 제외: 수학, 영어, 피아노, 태권도, 주방, 배달, 청소...

**Level 2: Claude Haiku AI 등급 분류**
| 등급 | 조건 | 액션 |
|------|------|------|
| S급 | 10만+/건 or 5만+/시간 + AI/코딩 + 온라인 | 즉시 지원 |
| A급 | 3만+/건 or 2만+/시간 + 기업/업셀 가능 | 검토 후 지원 |
| 패스 | 과외, 단순노무, 현장 | 스킵 |

### 숨고 전략 (계획중)
- "만원 폭탄" 전술: 5천원 견적 요청 → 1-2만원 견적 제출
- 차별화: "AI로 빠르게" 메시징
- 업셀링: 초기 접촉 후 추가 기능 제안

### 자동 지원
- ADB (Android Debug Bridge) 자동화 예정
- 타겟: Samsung S24+, 당근마켓 앱
- 지원 템플릿 + 이력 관리

## 기술 스택
| 영역 | 기술 |
|------|------|
| 스캐닝 | Python 3 (asyncio, httpx) |
| AI 분류 | Claude Haiku (빠르고 저렴) |
| 키워드 | Regex 패턴 매칭 |
| 데이터 | JSON (seen_jobs.json, scan_*.json) |
| 자동화 | ADB (Android Debug Bridge) 예정 |
| API | 당근마켓 Remix Data (인증 불필요) |
