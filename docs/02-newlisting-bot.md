# Newlisting Bot - 바이낸스 신규 상장 숏 트레이딩 봇

## 한줄 요약
바이낸스 선물 신규 상장 코인을 실시간 감지하고, 통계 기반 숏 전략으로 자동 매매하는 24/7 트레이딩 봇

## 프로젝트 개요
- **거래소**: 바이낸스 선물 (USDT-M, Isolated)
- **전략**: VWAP 기반 숏 (V_VWAP_1h, V_VWAP_24h)
- **현재**: V5 라이브 운영중 ($130 시드)
- **백테스트**: APY 96.5%, MDD 17.5%

## 핵심 기능

### 실시간 신규 상장 감지
- 바이낸스 exchangeInfo 10초 간격 폴링
- ~543 심볼 추적, 영속 메모리
- 신규 감지 시 제로 레이턴시 콜백

### 멀티 전략 트레이딩 엔진
| 전략 | 관찰시간 | SL/TP | 최대보유 |
|------|---------|-------|---------|
| V_VWAP_1h | 1시간 | 15%/15% | 48시간 |
| V_VWAP_24h | 24시간 | 15%/20% | 48시간 |

### 리스크 관리 (Kill Switch)
- 일일 손실 > 20% → 자동 셧다운
- 잔고 < $5 → 긴급 정지
- 최대 드로다운 > 40% → 정지
- 60분 쿨다운 후 재개

### 백테스팅 엔진
- Walk-Forward 검증 (3-fold 교차검증)
- 슬리피지, 펀딩비, 수수료 반영
- 700+ 파라미터 조합 그리드 서치

### 실시간 모니터링 대시보드 (React)
- Monitor: 실시간 봇 상태, 포지션, P&L
- Backtest: 전략 성과 지표
- Strategy Lab: 4단계 검증 파이프라인
- Research: 393개 코인 통계 분석
- Loops: 19회 자율 리서치 루프 기록

### 자율 리서치 루프
- 19회 리서치 루프, 30+ AI 에이전트 탐색
- 46개 전략 발굴 → 5개 백테스트 → 1개만 생존 (현재 VWAP)
- 크로스 거래소 차익, MEV, 옵션 등 심층 분석

## 핵심 인사이트 (개발 과정)
1. "주관적 스코어링은 완전히 실패" — 46전략 중 1개만 생존 (2.2%)
2. "비용이 전략을 죽인다" — 펀딩비 차익 이론 16-20% → 실제 -1.9%
3. "과다 레버리지는 파산" — 150% 노출 MDD 73% → 20%로 축소
4. "백테스트 수익은 63% 축소" — 이론 +1.91% → 실제 +0.71%
