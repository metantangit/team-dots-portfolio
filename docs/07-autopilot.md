# Autopilot - 자율 에이전트 미션 실행 시스템

## 한줄 요약
부동산 프로젝트의 다단계 미션을 AI 에이전트가 자율 실행하고, 단계별 사람 승인을 거치는 오케스트레이션 시스템

## 프로젝트 개요
- **버전**: v0.1 MVP
- **현재 미션**: 자양동 상가 임차인 영업 도구 자동 생성
- **승인 방식**: CLI 기반 (y/n/피드백)
- **의존성**: npm 패키지 제로 (순수 Node.js)

## 핵심 기능

### 미션 기반 실행 프레임워크
- projects.json 프로젝트 레지스트리
- mission.json 목표/파이프라인/타겟 정의
- 4단계 순차 실행 + 승인 게이트

### 4단계 파이프라인 (임차인 영업)
1. **분석**: 물건 스펙 → Claude AI → 5-7개 업종 추천
2. **연락처 리서치**: 추천 업종 → 담당자/연락처 조사
3. **영업자료 생성**: 맞춤형 제안서, 이메일, PDF
4. **배포**: Git 커밋 & 푸시

### 승인 & 피드백 루프
- 단계별 승인 인터페이스 (y/n/피드백)
- 최대 3회 재시도 (피드백 자동 반영)
- 승인 히스토리 context.json 저장
- 피드백 → 다음 Claude 프롬프트에 자동 주입

### 컨텍스트 영속성
- 프로젝트 의사결정 이력 누적
- 승인/거절 타임스탬프 기록
- 이전 결과 → 다음 단계 참조

## 기술 스택
| 영역 | 기술 |
|------|------|
| 런타임 | Node.js (ES Modules) |
| AI | Claude CLI (Sonnet) |
| 상태 | JSON 파일 (zero DB) |
| 의존성 | 제로 npm 패키지 |
| VCS | Git (배포 자동화) |

## 아키텍처
```
agent.js (오케스트레이터)
├── lib/claude.js      — Claude API 래퍼 (stdin/stdout)
├── lib/mission.js     — 미션/컨텍스트 파일 I/O
├── lib/approval.js    — CLI 승인 인터페이스 (한국어 IME 지원)
└── lib/steps/
    └── step1-analysis.js — Claude 기반 업종 분석
```
