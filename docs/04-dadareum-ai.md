# 다다름 FNC AI 플랫폼 - 프랜차이즈 운영 AI 자동화

## 한줄 요약
삼겹살 BBQ 프랜차이즈 (8개 매장, 연매출 ~800억) 운영 전반을 AI로 자동화하는 풀스택 플랫폼

## 프로젝트 개요
- **고객**: 다다름 FNC (제주돗날/제주묵돈 브랜드)
- **규모**: 8개 매장, 직원 ~50명
- **목표**: 90분 라이브 데모 (2026.02.24)
- **5대 모듈**: 스케줄 최적화, 메뉴 개발, CCTV 분석, 전략 로드맵, 퀀텀점프 시뮬레이터

## 5대 핵심 모듈

### 1. 자동 스케줄 최적화 (70% 완성)
- 4가지 시나리오 자동 생성 (최소/균형/여유/비상)
- 드래그앤드롭 주간/일간 스케줄링
- 포지션 적합도 매트릭스 (직원 스킬 ↔ 역할 매칭)
- 인건비 자동 계산 (급여 × 1.1733 4대보험)
- 주간 시나리오 비교
- 데이터: 7명 직원, POS 매출 데이터, 포지션 목표

### 2. 메뉴 개발 랩 (90% 완성)
- 현재 메뉴 분석 (반찬 12종)
- AI 신메뉴 제안 (원가/마진 분석)
- 레시피 상세 + 재료비 산출
- 계절성 평가 + 난이도 등급
- 벤치마크 레스토랑 비교

### 3. CCTV 동선 분석 (80% 완성)
- 실제 CCTV 영상 + 웹캠 캡처
- YOLOv8 객체 감지 (사람)
- 동선 히트맵 (고객/직원 분리)
- 구역별 병목 감지
- KPI: 전환율, 구역 커버리지
- 212초 촬영 영상 프레임별 분석

### 4. 전략 로드맵 (45% 완성)
- 손익계산서 파싱 (Excel → 시각화)
- 매출/비용 분해 시각화
- 시나리오 시뮬레이션 (현재 vs 개선)
- 계절성 분석 + 손익분기점 계산
- 리스크 시그널 감지
- 대상 매장: 제주돗날

### 5. 퀀텀점프 시뮬레이터 (40% 완성)
- 연간 손익 프로젝션
- 매출 범위 밴드 시각화
- 목표 역산 (목표 매출 → 필요 지표)
- 시나리오 비교 (보수/목표/공격)
- 월별/분기별 분해
- 대상 매장: 제주묵돈

## 기술 스택
| 영역 | 기술 |
|------|------|
| 프론트엔드 | Next.js 16, React 19, TypeScript, Tailwind CSS 4, shadcn/ui (32종) |
| 상태관리 | Zustand |
| 차트 | Recharts, Plotly.js |
| 백엔드 | FastAPI, SQLAlchemy (async), PostgreSQL 16, Redis 7 |
| AI/ML | LangChain + LangGraph, Ollama (exaone3.5:7.8b), YOLOv8 |
| 최적화 | OR-Tools (스케줄링 솔버) |
| 인프라 | Docker Compose, Makefile |
| 스크래핑 | Naver 리뷰/키워드 순위 |
