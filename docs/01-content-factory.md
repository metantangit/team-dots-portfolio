# Content Factory - AI 유튜브 콘텐츠 자동화 파이프라인

## 한줄 요약
AI 기반 유튜브 영상 제작 전 과정(리서치→대본→TTS→렌더링→배포)을 자동화하는 콘텐츠 팩토리

## 프로젝트 개요
- **채널명**: 빌릭 AI (Billik AI)
- **마스코트**: 샤크냥 (상어모자 쓴 고양이)
- **타겟**: 한국 개발자/AI 관심층
- **확장 도메인**: 자영업 AI, 부동산 자동화, DeFi/크립토

## 핵심 기능

### 8단계 콘텐츠 파이프라인
```
공정 0: 리서치 → 공정 1: 주제 선정 → 공정 2: 대본 작성
→ 공정 3: TTS 생성 → 공정 4: 비주얼 렌더링
→ 공정 5: 패키징 → 공정 6: 배포 → 공정 7: 분석
```

### 대시보드 (Next.js)
- KPI 카드: 총 영상수, 제작중, 게시완료
- 파이프라인 시각화 (실시간 상태)
- 대본 프리뷰 + 비주얼 태그 매핑
- TTS 비교 (A/B 테스트 3종 음성)
- 디자인 컨셉보드 (59+ 템플릿)

### TTS 음성 합성 (로컬)
- **모델**: Qwen3-TTS 1.7B (MLX, Apple Silicon 최적화)
- 음성 클로닝 (레퍼런스 오디오 기반)
- 속도/온도 컨트롤
- 세그먼트당 ~3분 생성 (M3)

### 비주얼 렌더링 (Remotion)
- React 컴포넌트 → MP4 변환
- 59+ 재사용 템플릿 (CSS/Motion/Drawing/3D)
- 30fps, 1280x720 출력

### 캐릭터 디자인 시스템
- 샤크냥 SVG + Zdog 3D + HTML 목업
- 컬러 시스템: #7DD3FC(메인), #FECDD3(악센트)
- 다크 테마 기반 (#0F0F0F)

## 기술 스택
| 영역 | 기술 |
|------|------|
| 대시보드 | Next.js 15, React 19, TypeScript, Tailwind CSS 4 |
| 애니메이션 | Framer Motion, GSAP, React Spring, Anime.js |
| 3D/그래픽 | Three.js, React Three Fiber, Konva, Zdog, Lottie |
| 영상 렌더링 | Remotion 4.0 |
| TTS | Qwen3-TTS (MLX), Whisper (타이밍 추출) |
| 자동화 | Python 3, Playwright (프리뷰 캡처) |
